English glossary



1. XAMARIN:

It is an open source platform for building modern and better performing applications for iOS, Android and Windows with . NET. Xamarin is an abstraction layer that manages the communication of shared code with the underlying platform code. (Microsoft,2020)

2. SQL:

It is a computer language for working with data sets and the relationships between them. Relational database programs, such as Microsoft Office Access, use SQL to work with data. (Datademia, 2012)

3. Json:

They are strings - useful when you want to transmit data over a network. It must be converted to a native JavaScript object when access to its data is required. This is not a problem, since JavaScript has a global JSON object that has methods available to convert between them. (Next_U, 2017)

Imagen 3



4. JSON WEB TOKEN:

It is an open standard based on JSON proposed by the IETF for the creation of access tokens that allow the propagation of identity and privileges or claims in English. (OpenWebinars, 2019)

Imagen 4



5. API:

The application programming interface, also known by the acronym API, in English, application programming interface, is a set of subroutines, functions and procedures that offers a certain library to be used by other software as an abstraction layer. (Xataka Basics, 2016)

Imagen 5



6. Controller:

Contains the flow control logic for an ASP.NET MVC application. A controller determines what response to return to a user when a user makes a browser request. A controller is simply a class (for example, a Visual Basic or C# class). (Microsoft,2015)

Imagen 6



7. Relational database:

They are based on the relational model, an intuitive and direct way of representing data in tables. In a relational database, each row in a table is a record with a unique ID, called a key. The columns of the table contain the attributes of the data, and each record typically has a value for each attribute, which makes it easy to create relationships between the data points. (Azure,2018)

Imagen 7




8. C#:

Is a multi-paradigm programming language developed and standardized by the Microsoft company as part of its .NET platform, which was later approved as a standard by the ECMA-334 (ECMA-334) and ISO (ISO/IEC 23270). C# is one of the programming languages designed for the common language framework. (Wikipedia,2022)

Imagen 8



9. MVC:

Model View Controller (MVC) is a style of software architecture that separates an application's data, user interface, and control logic into three distinct components. It is a very mature model that has proven its validity over the years in all kinds of applications, and on many languages and development platforms. (mdn web docs, 2014)

Imagen 9



10. Visual Studio:

It is an integrated development environment (IDE, for its acronym in English) for Windows and macOS. It supports multiple programming languages, such as C++, C#, Visual Basic .NET, F#, Java, Python, Ruby, and PHP, as well as web development environments, such as ASP.NET MVC, Django, etc. Add to that the new online capabilities under Windows Azure in the form of the Monaco editor. (Wikipedia,2022)

11. Scrum Methodology

It is a technique of the extended agile methodology that offers a way to connect various teams that need to work together to offer complex solutions. (Atlassian,2018)

Imagen 11


12. Sprint:

It is a short fixed period of time in which a srum team works to complete a set amount of work. (Atlassian,2018)

Imagen 12


13. Backlog:

The English word Backlog means “accumulation of something, especially incomplete work or things that we need to take care of”. (Muy agile,2019)

Imagen 13


14. Jira software:

It is part of a range of products designed to help teams of all kinds manage work. Jira was originally designed as a bug and issue tracker. However, it has become a powerful work management tool for all kinds of use cases, from requirements and test case management to agile software development. (Atlassian,2018)

Imagen 14



15. Framework:

It is software with a set of customizable and reusable components to build and maintain a website. Its name comes from English, which in Spanish means "structure", and refers to the base it provides for web development. In it there are libraries, extensions, plugins and other functionalities. (Wikipedia,2022)
